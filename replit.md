# Merced Trading Bot (MTB)

AI-powered French-language trading signal platform. Analyses real OHLC market data across 5 timeframes (M1/M5/M15/H1/H4) using RSI, EMA, MACD, ATR, and market structure — then pushes high-confidence trading signals (≥88%) with auto-calculated SL and TP1/TP2/TP3.

## Architecture

pnpm monorepo with path-based artifact routing through the Replit proxy.

```
lib/
  db/               — Drizzle ORM + PostgreSQL schema (users, sessions, trading accounts, signals, portfolio, push subscriptions)
  api-spec/         — OpenAPI 3 spec (source of truth for all routes)
  api-zod/          — Zod schemas auto-generated from OpenAPI spec
  api-client-react/ — TanStack Query hooks auto-generated by Orval

artifacts/
  api-server/       — Express 5 + TypeScript backend (port 8080, preview path /api-server)
  merced-trading-bot/ — React 19 + Vite 7 + TailwindCSS 4 frontend (port 24305, preview path /)
  mockup-sandbox/   — Vite design sandbox for Canvas prototyping
```

## Key Technical Decisions

- **Signal confidence**: `maxScore = h4 ? 127 : 110` — context-aware normalization so H4-unavailable paths can still emit signals at ≥88%
- **Cookies in production**: `secure: true, sameSite: "none"` (Replit HTTPS proxy); development: `secure: false, sameSite: "lax"`
- **CORS in production**: restricted to `*.replit.app`, `*.replit.dev`, `*.repl.co` patterns; open in dev
- **PWA strategy**: `injectManifest` — custom `src/sw.ts` compiled by Vite PWA; runtime caching defined inside sw.ts (NOT in vite.config workbox property)
- **Session table**: created via raw SQL in `ensureAllTables()` on startup (connect-pg-simple `createTableIfMissing` silently fails)
- **API errors**: `ApiError.data.error` contains the French error message from the backend

## User Preferences

- All user-facing copy is in **French** — do not add English strings to the frontend
- Keep the dark fintech aesthetic: `bg-background` dark, `text-accent` green (`#22c55e`), monospace labels
- Signal confidence threshold is ≥88% — do not lower it without explicit approval
- Real market data only — no mocked/random prices in signals
